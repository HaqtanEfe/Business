# assets/

- `hue.png` — the Hue bot avatar shown in the "Client experience" Discord mock.
  Drop your image here as **hue.png** (square works best, any size; it's shown at 36px, circular).
  If it's missing, the site falls back to an "H" monogram. To use a different filename/path,
  change `hueAvatar` in `../content.js`.

- `reviews/` — optional override images for client review avatars (e.g. studio logos).
  Reference one from `../content.js`, e.g. `avatar: "assets/reviews/studioname.png"`.
